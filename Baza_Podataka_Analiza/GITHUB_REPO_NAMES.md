```markdown
# Cineverse Database Model

## Overview
This repository contains the complete database design for a movie management system called Cineverse.

## Contents
- Conceptual ER diagram (PNG)
- Logical model with data types (PNG)
- SQL DDL scripts for database creation
- Sample queries and analysis
- Detailed table specifications

## Database Structure
- 7 normalized tables (3NF)
- 1:N and M:N relationships
- Movies, Directors, Genres, Actors, Countries

## Technologies
- MySQL/MariaDB
- Mermaid diagrams
- Markdown documentation
```
